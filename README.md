# SafeAtHome Guide

A consumer-facing directory and buyer's guide for aging-in-place home safety products and services.

## What It Is

SafeAtHome Guide covers products (stairlifts, walk-in tubs, grab bars), local CAPS-credentialed contractors, cost guides, and comparison content. Monetizes via affiliate commissions, lead sales, and contractor subscription listings.

## Stack

- **Frontend:** Next.js (Vercel)
- **Database:** Supabase (Postgres)
- **Email:** Resend
- **Payments:** Stripe

## Platform

Runs on a Mac Mini AI venture platform with OpenClaw orchestration, Slack command interface, and n8n workflow automation.
